# Grooming Salon Test

Тестовий односторінковий сайт для грумінг-салону.

## Що всередині

- `index.html` — структура сторінки
- `styles.css` — стилі та адаптивність
- `script.js` — базова логіка форми

## Локальний запуск

Відкрити `index.html` у браузері або підняти локальний сервер:

```bash
python3 -m http.server 8008
```

Потім відкрити:

```text
http://127.0.0.1:8008
```
